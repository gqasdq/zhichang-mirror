## 2. 模块功能树

### 2.1 情绪急救站（优先级：P0 - MVP核心）

**模块目标**：帮助求职者在焦虑、迷茫、挫败时快速获得情绪支持和行动指引

#### 功能树结构

```
情绪急救站 (EmotionFirstAid)
│
├── 子功能1: 情绪识别 (EmotionRecognition) [P0]
│   │
│   ├── 操作1.1: 文本情绪分析 [P0]
│   │   ├── 输入: 用户自由文本（任意长度）
│   │   ├── 处理: NLP情绪分类 + 强度评估
│   │   └── 输出: {"情绪类型", "情绪强度(1-10)", "核心关键词"}
│   │
│   ├── 操作1.2: 预设标签选择 [P0]
│   │   ├── 输入: 预设情绪标签列表
│   │   ├── 处理: 用户点击选择
│   │   └── 输出: {"情绪类型", "情绪来源"}
│   │
│   └── 操作1.3: 情绪画像生成 [P1]
│       ├── 输入: 多轮对话历史
│       ├── 处理: 综合情绪分析
│       └── 输出: 情绪画像报告
│
├── 子功能2: 同路人匹配 (PeerMatching) [P0]
│   │
│   ├── 操作2.1: 故事检索 [P0]
│   │   ├── 输入: 情绪类型 + 行业 + 岗位级别
│   │   ├── 处理: FAISS语义检索 + 规则过滤
│   │   └── 输出: Top3相似故事摘要
│   │
│   ├── 操作2.2: 故事详情展示 [P0]
│   │   ├── 输入: 故事ID
│   │   ├── 处理: SQLite查询 + 格式化
│   │   └── 输出: 完整故事卡片
│   │
│   └── 操作2.3: 相似度评分 [P1]
│       ├── 输入: 用户画像 vs 故事主角画像
│       ├── 处理: 多维度相似度计算
│       └── 输出: 相似度百分比
│
├── 子功能3: 情感回应 (EmpathicResponse) [P0]
│   │
│   ├── 操作3.1: 即时共情回应 [P0]
│   │   ├── 输入: 情绪类型 + 情绪强度
│   │   ├── 处理: 模板匹配 + LLM润色
│   │   └── 输出: 共情文本 (100-200字)
│   │
│   ├── 操作3.2: 分层回应策略 [P1]
│   │   ├── 输入: 情绪强度
│   │   ├── 处理: 分层策略选择
│   │   └── 输出: 不同深度的回应
│   │
│   └── 操作3.3: 个性化调适 [P2]
│       ├── 输入: 用户偏好设置
│       ├── 处理: 风格调适
│       └── 输出: 符合用户风格的回应
│
└── 子功能4: 行动引导 (ActionGuidance) [P0]
    │
    ├── 操作4.1: 即时行动建议 [P0]
    │   ├── 输入: 情绪类型 + 当前状态
    │   ├── 处理: 规则匹配 + 优先级排序
    │   └── 输出: 3-5个可执行行动建议
    │
    ├── 操作4.2: 资源推荐 [P1]
    │   ├── 输入: 情绪类型 + 用户画像
    │   ├── 处理: 知识库检索
    │   └── 输出: 推荐资源列表
    │
    └── 操作4.3: 后续跟进提醒 [P2]
        ├── 输入: 行动计划
        ├── 处理: 提醒调度
        └── 输出: 定时提醒设置
```

#### MVP包含功能

- [x] 情绪识别 (文本分析 + 预设标签)
- [x] 同路人匹配 (故事检索 + 详情展示)
- [x] 情感回应 (即时共情)
- [x] 行动引导 (即时建议)

#### 后续迭代

- [ ] 情绪画像生成
- [ ] 分层回应策略
- [ ] 资源推荐
- [ ] 后续跟进提醒

### 2.2 金子探测器（优先级：P0 - MVP核心）

**模块目标**：帮助用户发现简历中的亮点，检测可能被埋没的价值

#### 功能树结构

```
金子探测器 (ResumeGoldFinder)
│
├── 子功能1: 简历解析 (ResumeParser) [P0]
│   │
│   ├── 操作1.1: 文本简历解析 [P0]
│   │   ├── 输入: 用户粘贴的简历文本
│   │   ├── 处理: 结构化字段提取 + 关键信息识别
│   │   └── 输出: {"姓名","联系方式","教育背景","工作经历","项目经验","技能清单"}
│   │
│   ├── 操作1.2: 简历上传解析 [P0]
│   │   ├── 输入: PDF/Word文件
│   │   ├── 处理: 文件解析 + 文本提取
│   │   └── 输出: 结构化简历数据
│   │
│   ├── 操作1.3: 字段完整性检测 [P1]
│   │   ├── 输入: 结构化简历数据
│   │   ├── 处理: 字段检测
│   │   └── 输出: 缺失字段列表
│   │
│   └── 操作1.4: 多版本管理 [P2]
│       ├── 输入: 用户操作
│       ├── 处理: 版本控制
│       └── 输出: 版本历史
│
├── 子功能2: 偏见检测 (BiasDetector) [P0]
│   │
│   ├── 操作2.1: ATS关键词检测 [P0]
│   │   ├── 输入: 简历文本 + 目标岗位
│   │   ├── 处理: 关键词匹配 + 覆盖率计算
│   │   └── 输出: {"关键词覆盖率", "缺失关键词列表"}
│   │
│   ├── 操作2.2: 价值低估预警 [P0]
│   │   ├── 输入: 工作经历描述
│   │   ├── 处理: 价值挖掘算法
│   │   └── 输出: {"低估点", "原描述", "建议表述"}
│   │
│   ├── 操作2.3: 偏见类型识别 [P1]
│   │   ├── 输入: 简历内容
│   │   ├── 处理: 偏见规则检测
│   │   └── 输出: 偏见类型列表 + 风险等级
│   │
│   └── 操作2.4: 行业基准对比 [P2]
│       ├── 输入: 简历内容 + 行业选择
│       ├── 处理: 行业数据对比
│       └── 输出: 对比报告
│
└── 子功能3: 翻案建议 (RevivalSuggestion) [P0]
    │
    ├── 操作3.1: 简历优化建议 [P0]
    │   ├── 输入: 简历解析结果 + 偏见检测结果
    │   ├── 处理: LLM生成优化建议
    │   └── 输出: 分点优化建议列表
    │
    ├── 操作3.2: 亮点重写 [P0]
    │   ├── 输入: 低估描述
    │   ├── 处理: LLM重写
    │   └── 输出: 重写后的描述
    │
    ├── 操作3.3: 关键词补充 [P1]
    │   ├── 输入: 缺失关键词 + 简历上下文
    │   ├── 处理: 智能插入建议
    │   └── 输出: 修改建议
    │
    └── 操作3.4: 简历生成 [P2]
        ├── 输入: 用户基础信息
        ├── 处理: 全自动生成
        └── 输出: 完整简历草稿
```

│
├── 子功能1: 选择理解 (ChoiceUnderstanding) [P1]
│   │
│   ├── 操作1.1: 选择输入解析 [P1]

### 4.2 状态定义与触发

| 状态ID               | 状态名称  | 进入条件     | 触发功能          | 退出条件   |
| ------------------ | ----- | -------- | ------------- | ------ |
| GUEST              | 访客    | 首次打开应用   | -             | 用户首次交互 |
| WELCOME            | 欢迎页   | 新用户首次使用  | 功能介绍          | 选择功能模块 |
| EMOTION\_AID       | 情绪急救站 | 点击情绪入口   | 情绪识别→匹配→回应→行动 | 完成或返回  |
| RESUME             | 金子探测器 | 点击简历入口   | 解析→检测→建议      | 完成或返回  |
| PARALLEL\_UNIVERSE | 平行宇宙  | 点击平行宇宙入口 | 选择理解→推演→对比    | 完成或返回  |
| GENE\_SEQUENCE     | 基因测序  | 点击基因测序入口 | 场景问答→分析→报告    | 完成或返回  |
| EMPATHY\_CHAIN     | 人才共情链 | 点击共情链入口  | 路线图展示         | 完成或返回  |
| PROCESSING         | 处理中   | 发起任何请求   | Agent调度       | 结果返回   |
| RESULT             | 结果展示  | 处理完成     | 结果渲染          | 用户操作   |
| CONTINUE           | 继续    | 结果页操作    | -             | 选择后续操作 |
| MEMORY             | 记忆更新  | 退出任何功能   | 数据持久化         | 更新完成   |
